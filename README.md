# 🧮 Calculator App — Python

A professional desktop calculator built with Python and Tkinter.

## Features
- Clean dark-themed UI (iOS style)
- Basic arithmetic: + − × ÷
- Percentage and negate support
- Keyboard shortcuts
- Auto font scaling

## How to Run
```bash
python calculator.py
```

## Tech Used
- Python 3
- Tkinter (built-in)
